# 對戰體驗優化 Phase 4 — 體驗打磨

**版本：** 1.0
**日期：** 2026-05-11
**狀態：** 已批准

---

## 1. Hit freeze / 打擊停頓

命中敵人時，遊戲短暫停頓 3-5 幀，增強打擊感。

| 事件 | 停頓長度 |
|------|----------|
| 普通攻擊命中 | 50ms |
| 技能命中 | 80ms |
| 暴擊 | 120ms |
| 絕招命中 | 150ms |

透過 `this.time.timeScale` 或暫停 tween/物理來實作。

## 2. ATB 行動序列預覽

畫面右側顯示接下來 3 個單位的行動順序，讓玩家可以提前規劃。

設計：在畫面右側垂直排列 3 個頭像/名稱，由上至下表示行動順序。

## 3. 戰鬥轉場特效

目前進入/離開戰鬥只有簡單 fade。改進為：
- 進入戰鬥：畫面扭曲收縮+放大（使用 Phaser 相機效果）
- 離開戰鬥：勝利用金色 flash，戰敗用紅色 flash
- 遭遇精英/頭目時：相機 shake + 特殊文字提示

## 4. 成就／稱號系統

輕量級成就系統，追蹤玩家里程碑：

| 成就 | 條件 | 獎勵 |
|------|------|------|
| 初出茅廬 | 完成第一次戰鬥 | 100 銀兩 |
| 百戰勇士 | 戰鬥 10 次 | 草藥水×3 |
| 千人斬 | 擊敗 100 個敵人 | 稱號「千人斬」 |
| 裝備收藏家 | 獲得 5 件不同裝備 | 戒指 |
| 技能大師 | 點滿一個武功的技能樹 | 玉珮 |
