## 1. Phase A — 江湖歷練核心

### 1.1 五圍屬性系統

| 屬性 | 英文 | 影響 |
|------|------|------|
| 臂力 | STR | 物理攻擊力 |
| 膽識 | BRA | 暴擊率、特殊效果觸發率 |
| 悟性 | WIS | 學點消耗折扣、武功升級速度 |
| 福緣 | LUK | 稀有掉落率、奇遇觸發率 |
| 定力 | CON | 防禦力、狀態抗性 |

創角配點：初始 50 點自由分配，單項下限 5、上限 15。

### 1.2 實戰經驗與學點

```
實戰升級需求 = level × 200
每級加成：HP +20、MP +10
學點獲得 = 戰鬥 EXP × 0.5（例：打一場得 60 EXP → 30 學點）
武功學習/升級消耗學點（詳見 Phase B）
```

### 1.3 名聲與善惡值

| 行為 | 名聲變化 | 善惡變化 |
|------|---------|---------|
| 戰鬥勝利 | +2 | 0 |
| 擊敗頭目 | +30 | 0 |
| 完成俠義任務 | +5~20 | +10~30 |
| 殺 NPC | -10 | -50 |
| 叛派 | -50 | -100 |

善惡值範圍 -1000 ~ 1000，影響門派加入條件與 NPC 對話內容。

### 1.4 角色面板 UI

按 C 鍵開啟，顯示五圍、實戰、學點、名聲、善惡（附標籤：大俠/俠士/中立/惡徒/魔頭）、門派、裝備欄位。

## 2. Phase B — 門派與武功

```
遊戲啟動 → 輸入名稱 → 分配 50 點五圍屬性（拖曳 +/- 按鈕）

    → 自由探索，透過 NPC 了解門派 → 決定加入
```

取代目前的固定角色選擇場景。

### 2.3 武功學習規則

```
找門派師父 NPC → 選擇「學習武功」→ 顯示可學列表（依階級過濾）
    → 消耗學點學習 → 加入 martialArts 列表

學點消耗：
  初階武功：50 學點
  中階武功：150 學點 + 門派聲望 ≥ 500
  絕學武功：400 學點 + 門派聲望 ≥ 2000 + 職位護法以上
```

武功升級（找師父或使用面板）：
```
LV2：消耗 30 學點 + 100 實戰
LV3：消耗 80 學點 + 300 實戰
LV4：消耗 150 學點 + 600 實戰
LV5（Max）：消耗 300 學點 + 1000 實戰
```

### 2.4 武功裝備欄

- 4 格裝備欄位，自由裝配已學武功
- 戰鬥中按 1-4 使用
- 可在非戰鬥狀態下隨時更換配置

### 2.5 叛派機制

找特定 NPC 選擇叛派：
- 該門派所有武功遺忘
- 門派聲望歸零
- 善惡值 -100
- 少林叛出後不可再回少林

## 3. Phase C — 任務系統

### 3.1 NPC 對話框架

走近 NPC → 按 E → 對話選單：

```
[NPC 名稱] - [稱號]
├─ 交談
├─ 學習武功（門派師父）
├─ 交易（商人）
├─ 捐獻（門派 NPC）
└─ 接取任務（任務 NPC）
```

### 3.2 任務類型

| 類型 | 內容 | 獎勵 |
|------|------|------|
| 鏢局護送 | 從 A 地護送鏢物到 B 地，途中隨機遇敵 | 銀兩 + 名聲 |
| 賞金獵人 | 擊殺指定數量的某種敵人 | 銀兩 + 學點 |
| 門派任務 | 收集材料、巡邏、清理門派區域敵人 | 門派聲望 |
| 劇情任務 | 連鎖事件，有故事線 | 武功秘笈 + 大量經驗 |

### 3.3 任務資料結構

```js
{ id, title, desc, type, objectives: [{type: 'kill'|'collect'|'escort', target, count, current}],
  rewards: { silver, exp, studyPoints, item, fame }, isComplete: false }
```

### 3.4 任務追蹤 UI

按 J 開啟任務面板，顯示已接任務列表、當前進度、目標地點提示。最多同時持有 5 個任務。

## 4. Phase D — 自創武功

### 4.1 融合條件

- 學習至少 8 套不同門派的武功
- 有至少 2 套武功達到 LV3 以上
- 找特定 NPC「隱士高人」進行融合

### 4.2 融合規則

```
選擇 2 套武功作為基底（主 + 副）
外功 + 外功 → 物理系（高攻擊倍率）
內功 + 內功 → 內功系（高範圍/特殊效果）
外功 + 內功 → 均衡系（附帶額外效果）
絕學 + 絕學 → 有 30% 機率出「絕世武功」（全屬性大幅加成）
```

自創武功初始為 LV1，可消耗學點升級（同一般武功）。

### 4.3 UI

- 融合介面顯示兩格武功選擇欄
- 選定後顯示預覽效果
- 可自訂名稱（最多 8 字）
- 確認融合後消耗大量學點（500）

## 6. Phase 實作順序與相依

```
Phase A ─── 無依賴，最先實作
  │
Phase B ─── 依賴 Phase A（學點 + 屬性）
  │
  ├──→ Phase D ─── 依賴 Phase B（武功系統完成後）
  │
Phase C ─── 可與 Phase B 並行開發
  │
Phase E ─── 可最後實作，完全獨立
```
