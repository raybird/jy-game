## 2. ATB 半即時制（核心改造）

### 2.1 設計動機

目前 Phaser 戰鬥為純回合制（玩家 → 敵人 → 玩家），節奏單一且無法體現敏捷差異。ATB 讓不同角色有不同出手速度，小龍女（高 AGI）出手頻率是郭靖（低 AGI）的兩倍以上。

### 2.2 ATB 數值公式

```
ATB 充能速度 = 單位速度值 × delta × 0.001
速度值 = AGI × 1.5（玩家）/ 基礎速度 8-15（敵人）
ATB 上限 = 100
```

當 ATB ≥ 100 時，該單位進入待命狀態。系統從所有 ATB ≥ 100 的單位中選取最高者行動，行動後 ATB 歸零。

### 2.3 敵人的速度值

| 敵人 | 速度值 |
|------|--------|
| 全真弟子 | 8 |
| 道士 | 10 |
| 明教教徒 | 12 |
| 波斯人 | 15 |

### 2.4 戰鬥流程（ATB 版）

```
遊戲開始 → 雙方 ATB 開始填充
  │
  └→ 單位 A 的 ATB ≥ 100（且為最高）
       │
       ├→ A 是玩家 → 等待輸入 → 執行行動 → ATB歸零 → 繼續填充
       │
       └→ A 是敵人 → 自動行動（0.5s延遲） → ATB歸零 → 繼續填充
```

### 2.5 ATB UI

- 畫面頂部顯示敵我雙方的 ATB 進度條
- 進度條顏色：玩家為藍色、敵人為紅色
- 進度條上方顯示單位名稱
- 待命中的單位進度條閃爍
- 行動序列預覽：在 ATB 條旁顯示下一個行動的單位頭像或名稱

### 2.6 狀態對 ATB 的影響

- 封穴狀態：ATB 停止填充 1 回合
- 輕靈屬性相剋加成：ATB 充能速度 +20%
- 被動加成（小龍女技能樹）：ATB 充能速度 +15%

## 3. 武俠屬性相剋

### 3.1 四屬性循環

```
陰柔 ← 克制 → 陽剛 ← 克制 → 剛猛 ← 克制 → 輕靈 ← 克制 → 陰柔
```

### 3.2 角色屬性歸屬

| 角色 | 屬性 | 說明 |
|------|------|------|
| 郭靖 | 剛猛 | 降龍十八掌、左右互搏皆為剛猛系 |
| 楊過 | 陽剛 | 黯然銷魂掌、玄鐵劍法為陽剛系 |
| 小龍女 | 輕靈 | 玉女劍法、雙劍合璧為輕靈系 |
| 張無忌 | 陰柔 | 乾坤大挪移、九陽神功為陰柔系 |
| 令狐冲 | 無（全能） | 所有技能無屬性，不受克制也不克制他人 |

### 3.3 技能屬性對照

每個角色的 4 個技能中，前 2 個為角色主屬性，後 2 個為副屬性（或無屬性）：

| 角色 | 技能 1 | 技能 2 | 技能 3 | 技能 4 |
|------|--------|--------|--------|--------|
| 郭靖 | 剛猛 | 剛猛 | 剛猛 | 陰柔（九陰真經） |
| 楊過 | 陽剛 | 陽剛 | 輕靈（彈指神通） | 無（玉女心經） |
| 小龍女 | 輕靈 | 輕靈 | 輕靈 | 陰柔（冰魄銀針） |
| 張無忌 | 無（乾坤） | 剛猛（七傷拳） | 陽剛（聖火令） | 陰柔（九陽神功） |
| 令狐冲 | 無 | 無 | 無 | 無 |

### 3.4 相剋傷害公式

```
屬性加成 = 
  克制 target 時: 1.3
  被 target 克制時: 0.7
  無關時: 1.0
  令狐冲所有技能: 1.0（不受影響）

最終傷害 = 基礎傷害 × 屬性加成
```

### 3.5 UI 提示

- 技能按鈕旁顯示屬性圖示（⚡陽剛 ❄陰柔 ⛰剛猛 🍃輕靈）
- 攻擊命中時顯示屬性效果文字（「⚡陽剛 克制 陰柔！傷害 +30%」）
- 角色選擇畫面顯示角色屬性

## 4. 怒氣大招系統

### 4.1 怒氣機制

| 事件 | 怒氣獲得 |
|------|----------|
| 對敵人造成傷害 | +10 |
| 受到傷害 | +15 |
| 每回合被動 | +5 |
| 上限 | 100 |

### 4.2 角色專屬絕招

| 角色 | 絕招名稱 | 效果 | 動畫 |
|------|----------|------|------|
| 郭靖 | 亢龍有悔 | 300% 傷害，無視防禦 | 大型金色龍形掌印 |
| 楊過 | 玄鐵重劍 | 400% 傷害，自身損血 10% | 巨大黑色劍氣 |
| 小龍女 | 玉女素心劍 | 攻擊 3 次，每次 150% 傷害 | 白色劍氣連閃 |
| 張無忌 | 乾坤大挪移 | 反彈所有傷害 3 回合 + 恢復 30% HP | 金色護盾 |

- 怒氣滿 100 時，畫面中央顯示「按 R 釋放絕招！」提示

- 按 R 後播放專屬動畫，造成傷害後回到正常 ATB 流程

## 5. 招架／閃避

### 5.1 招架（主動防禦）

- 按 F 鍵進入招架狀態
- 招架狀態下：本回合受到的所有傷害 -50%
- 招架狀態持續到玩家下一次行動
- 招架時無法攻擊或使用技能
- 招架不消耗 MP

### 5.2 閃避（被動）

- 基於 AGI 的被動機率
- 閃避機率 = `AGI / (AGI + 50)`（小龍女 AGI=18 → 26.5%，郭靖 AGI=8 → 13.8%）
- 成功閃避時完全不受傷害
- 閃避觸發時顯示「miss」文字

### 5.3 招式與閃避的互動

- 獨孤九劍：無視閃避（`ignoreDodge: true`）
- 玄鐵劍法：無視防禦也無視閃避
- 封穴狀態下無法招架（但可被動閃避）

## 6. 裝備影響戰鬥

### 6.1 裝備欄位效果

| 欄位 | 效果 | 數值公式 |
|------|------|----------|
| 武器 | +攻擊力 | 裝備 `attack` 值直接加算到普攻/技能傷害 |
| 防具 | +防禦力 | 裝備 `defense` 值加算到角色防禦 |
| 飾品 | 特殊效果 | 依飾品類型不同 |

### 6.2 飾品效果類型

| 飾品 | 效果 |
|------|------|
| 玉珮 | 每回合恢復 HP 2% |
| 護身符 | 受到傷害 -5% |
| 戒指 | 暴擊率 +5% |
| 腰墜 | 怒氣獲得 +20% |

### 6.3 裝備資料結構

```js
// ITEMS 擴充
iron_sword: { name: '鐵劍', type: 'weapon', battleStats: { attack: 10 } }
leather_armor: { name: '皮甲', type: 'armor', battleStats: { defense: 5 } }
jade_pendant: { name: '玉珮', type: 'accessory', battleStats: { hpRegenCombat: 0.02 } }
```

### 6.4 傷害公式（整合版）

```
角色攻擊力 = 基礎攻擊 + 武器攻擊 + innerPower × 0.3
角色防禦力 = 基礎防禦（0）+ 防具防禦 + 飾品防禦buff

技能傷害 = 角色攻擊力 × 技能倍率 × 屬性加成 × 技能樹加成
最終傷害 = max(1, 技能傷害 - 角色防禦力 × 無視防禦係數)
```
