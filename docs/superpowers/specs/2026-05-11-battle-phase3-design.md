# 對戰體驗優化 Phase 3 — 內容擴充

**版本：** 1.0
**日期：** 2026-05-11
**狀態：** 已批准

## 1. 敵人分級

每個地圖的敵人分為三級：

| 分級 | HP 倍率 | 攻擊倍率 | 防禦倍率 | EXP | 銀兩 | 掉落 |
|------|---------|---------|---------|-----|------|------|
| 普通 | ×1.0 | ×1.0 | ×1.0 | 60 | 80 | 一般 |
| 精英 | ×2.0 | ×1.5 | ×1.3 | 150 | 200 | 一般 + 稀有 |
| 頭目 | ×3.0 | ×2.0 | ×1.5 | 300 | 500 | 保證稀有 |

精英/頭目名稱前綴：「精英」+原名、「頭目」+原名。

精英出現率 20%，頭目 5%。戰鬥前隨機決定分級。

## 2. 多樣化戰利品

每個敵人獨立的掉落表：

| 敵人 | 普通掉落 (40%) | 稀有掉落 (15%) |
|------|---------------|---------------|
| 全真弟子 | iron_ore×1 | herb×2 |
| 道士 | herb×2 | herb_potion×1 |
| 明教教徒 | bronze_ore×1 | iron_sword 圖紙 |
| 波斯人 | leather×2 | leather_armor 圖紙 |

精英追加保證一個稀有掉落。頭目追加保證兩個稀有掉落 + 飾品。

## 3. 角色專屬技能動畫

目前的 `skillAnimation(index)` 使用通用彩色光球。改為每個角色不同：

| 角色 | 動畫效果 |
|------|----------|
| 郭靖 | 金色大型掌印（降龍十八掌）、棕色拳影（空明拳） |
| 楊過 | 暗紅色劍氣（玄鐵劍法）、黑色衝擊波（黯然銷魂掌） |
| 小龍女 | 白色劍氣連閃（玉女劍法）、冰藍色針影（冰魄銀針） |
| 張無忌 | 金色護盾波動（乾坤大挪移）、赤紅色拳印（七傷拳） |
| 令狐冲 | 藍色高速劍氣（獨孤九劍）、紫色吸力漩渦（吸星大法） |

使用 Phaser 幾何圖形 + tween 組合，不需外部 sprite。

## 4. 音效系統

使用 Web Audio API（OscillatorNode + GainNode）生成簡單音效，不需外部音檔：

| 事件 | 音效 |
|------|------|
| 普攻命中 | 短噪音 (100ms, 200Hz square) |
| 技能施放 | 上升音 (300ms, 300→600Hz sine) |
| 暴擊 | 高亢撞擊音 (200ms, 400Hz sawtooth) |
| 受擊 | 低沉撞擊音 (150ms, 150Hz triangle) |
| 勝利 | 上升三連音 (各100ms) |
| 戰敗 | 下降音 (500ms, 400→100Hz sine) |
| 升級 | 歡呼音 (400ms, 500→800Hz sine) |

封裝為 `SoundManager` 類別，全域 singleton，可在 BattleScene 和 WorldScene 中呼叫。
